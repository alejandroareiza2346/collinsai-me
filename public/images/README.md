# Imágenes del sitio

Carpeta para almacenar imágenes públicas usadas en la web de CollinsAI.

Estructura sugerida:

- `hero/` - imágenes para hero y banners.
- `team/` - fotos del equipo y avatars.
- `logos/` - logos de clientes o partners (SVG/PNG).
- `cases/` - imágenes usadas en casos de éxito (privados si procede).
- `blog/` - imágenes de posts y recursos.
- `demo/` - imágenes relacionadas con demos y mockups.
- `social/` - imágenes para compartir en redes (Open Graph).
- `backgrounds/` - patterns y fondos.
- `icons/` - iconografía y assets pequeños.

Guia rapida:

- Mantén nombres lowercase y con guiones (`mi-imagen.jpg`).
- Evita espacios y caracteres especiales.
- Si un archivo es temporal, añade `README` o `.gitkeep` y no subas archivos sensibles.
