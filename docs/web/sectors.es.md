# Sectores objetivo — CollinsAI

Nos enfocamos en PYMEs que operan con procesos manuales y necesitan escala operativa:

- Salud (clínicas, consultorios): gestión de citas, recordatorios, conciliación de pagos.
- Inmobiliario: gestión de leads, agendamiento de visitas, seguimiento de compradores.
- Ecommerce y retail: manejo de pedidos, notificaciones de envío, conciliación de inventario.
- Legal y contable: seguimiento de documentos, recordatorios de vencimientos, gestión de clientes.
- Servicios profesionales (consultorías, freelancers): facturación, seguimiento de propuestas, CRM ligero.

Para cada sector definimos: puntos de dolor, flujo mínimo viable de automatización, KPIs operativos y entregables estándar.
