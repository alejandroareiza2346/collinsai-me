# Servicios — CollinsAI (Fase 1: Agencia)

## Servicios principales

### Automatización de procesos operativos

- Mapeo de procesos, scripts y flujos n8n para eliminar tareas manuales.
- Ejemplos: conciliación de pagos, generación de facturas, sincronización de inventarios.

### CRM automatizado y seguimiento comercial

- Pipeline automatizado con triggers desde formularios, WhatsApp y correo.
- Tareas automáticas de contacto, recordatorios y puntuación de leads.

### IA conversacional para WhatsApp

- Bots de pre-calificación, respuestas frecuentes, escalamiento a humano y recordatorios.
- Integración segura con CRM y logs de auditoría.

### Dashboards operativos y reporting

- Conector a PostgreSQL y Redis; paneles con KPIs, alertas y reportes periódicos.

### Integraciones API y conectores

- Conexiones seguras con ERPs, pasarelas de pago y proveedores externos.

### Agentes IA para tareas operativas

- Agentes que automatizan acciones (ej. procesar órdenes, enviar notificaciones).

### Consultoría y seguridad

- Auditoría de procesos, gobernanza de datos y hardening de infra (HSTS, CSP, X-Frame-Options).

## Modelo de entrega

- Diagnóstico operativo → MVP de automatización (2–6 semanas) → Iteración y monitoreo.
- Entrega por proyecto con hitos medibles y reporte de impacto (tiempo ahorrado, leads recuperados).
